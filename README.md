# Welcome to FCEditor. 
### FCEditor is a 3D Unity level editor for the 1998 PC game "Future Cop LAPD".
### The editor is capable of modifying both Precinct Assault levels and Crime War levels.
### To learn more about the editor, check out the complete guide series on the editor: https://youtube.com/playlist?list=PLcRqyO43QxHc2E1mwR8464nuCxYaWbMFb&si=RJv30CUfLMsOL4kp
